# Serverless Cricket Overlay (Mobile Ready)

A lightweight, serverless-capable live cricket overlay for OBS/vMix, optimized for mobile streaming via Termux.

## Features
-   **Live Score Updates**: Fetches data from CricClubs.
-   **Dynamic League Support**: Automatically detects league and logo from URL.
-   **Mobile Streaming**: Run the server on your Android phone using Termux.
-   **Broadcast Design**: Professional glassmorphism UI.

## Quick Start (PC)
1.  Install Node.js.
2.  Run `npm install`.
3.  Run `node server.js`.
4.  Open `index.html` in your browser.
